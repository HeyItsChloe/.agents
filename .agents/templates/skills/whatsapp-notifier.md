# WhatsApp Notifier — {{REPO_NAME}}

Sends review request to your phone via WhatsApp (callmebot API).

## Process

### Step 1: Get PR Info

```bash
PR_NUMBER=$(cat /tmp/pr-number.txt 2>/dev/null || echo "")
PR_URL=$(cat /tmp/pr-url.txt 2>/dev/null || echo "")
CI_STATUS=${CI_STATUS:-"unknown"}

echo "PR #$PR_NUMBER: $PR_URL"
echo "CI Status: $CI_STATUS"
```

### Step 2: Prepare Notification Message

```bash
# Prepare message based on status
if [ "$CI_STATUS" == "passed" ]; then
  MESSAGE="✅ PR #$PR_NUMBER is ready for your review.
  
Link: $PR_URL
  
Generated by pipeline-generator for {{REPO_NAME}}"
elif [ "$CI_STATUS" == "failed" ]; then
  MESSAGE="❌ CI failed for PR #$PR_NUMBER.
  
Manual action required.
Link: $PR_URL
  
Check CI logs and fix issues."
else
  MESSAGE="📋 PR #$PR_NUMBER created for your review.
  
Status: CI pending
Link: $PR_URL
  
Generated by pipeline-generator for {{REPO_NAME}}"
fi

echo "Message:"
echo "$MESSAGE"
```

### Step 3: Send via callmebot API

```bash
# Using callmebot API (user must have sent initial message to +34 644 76 60 71)
WHATSAPP_PHONE=${WHATSAPP_PHONE:-""}
WHATSAPP_API_KEY=${WHATSAPP_API_KEY:-""}

if [ -n "$WHATSAPP_PHONE" ] && [ -n "$WHATSAPP_API_KEY" ]; then
  # Send message via callmebot
  curl -s "https://api.callmebot.com/text.php?phone=$WHATSAPP_PHONE&text=$(echo "$MESSAGE" | sed 's/ /%20/g' | sed 's/\n/%0A/g')&apikey=$WHATSAPP_API_KEY"
  
  echo ""
  echo "✅ WhatsApp notification sent to $WHATSAPP_PHONE"
else
  echo "⚠️ WhatsApp credentials not configured"
  echo "To enable notifications:"
  echo "1. Message +34 644 76 60 71: 'I allow callmebot to send me messages'"
  echo "2. Add WHATSAPP_PHONE and WHATSAPP_API_KEY to OpenHands secrets"
fi
```

### Step 4: Fallback - Post PR Comment

```bash
# If WhatsApp not configured, post to PR as fallback
PR_NUMBER=$(cat /tmp/pr-number.txt 2>/dev/null || echo "")

if [ -n "$PR_NUMBER" ]; then
  gh pr comment {{OWNER}}/{{REPO_NAME}}/$PR_NUMBER --body "
## 📱 Notification

$([ "$CI_STATUS" == "passed" ] && echo "✅ PR is ready for review" || echo "⚠️ CI status: $CI_STATUS")

PR: #$PR_NUMBER
Link: $PR_URL

$([ "$CI_STATUS" != "passed" ] && echo "Manual intervention may be required." || echo "Please review and merge when ready.")
" 2>/dev/null || echo "Could not post comment"
fi
```

## Alternative: Twilio

```bash
# Alternative: Twilio WhatsApp (if configured)
TWILIO_ACCOUNT_SID=${TWILIO_ACCOUNT_SID:-""}
TWILIO_AUTH_TOKEN=${TWILIO_AUTH_TOKEN:-""}
TWILIO_FROM=${TWILIO_FROM:-""}

if [ -n "$TWILIO_ACCOUNT_SID" ] && [ -n "$TWILIO_AUTH_TOKEN" ]; then
  curl -X POST "https://api.twilio.com/2010-04-01/Accounts/$TWILIO_ACCOUNT_SID/Messages.json" \
    -u "$TWILIO_ACCOUNT_SID:$TWILIO_AUTH_TOKEN" \
    -d "To=whatsapp:$WHATSAPP_PHONE" \
    -d "From=whatsapp:$TWILIO_FROM" \
    -d "Body=$MESSAGE"
fi
```

## Configuration Required

To enable WhatsApp notifications, add these secrets to OpenHands:

| Secret | How to get |
|--------|-----------|
| `WHATSAPP_PHONE` | Your number, international format, no + (e.g., 447911123456) |
| `WHATSAPP_API_KEY` | Message `+34 644 76 60 71`: `I allow callmebot to send me messages` |

## Error Handling

If notification fails:
1. Log error to `/tmp/whatsapp-notifier-error.log`
2. Continue pipeline (notification is non-critical)
3. PR comment serves as fallback

## Next Step

Pipeline complete. User reviews PR and merges manually.
