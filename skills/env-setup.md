# Environment Setup Skill

Writes config files from OpenHands secrets before run commands. Use this skill at the
start of any pipeline that needs authenticated access (GitHub API, external services).

## Required Secrets

Register in OpenHands → Settings → Secrets:

| Secret | Required | Description |
|--------|----------|-------------|
| `GITHUB_TOKEN` | Yes | GitHub personal access token for API access |
| `WHATSAPP_PHONE` | No | WhatsApp number for notifications (no `+`) |
| `WHATSAPP_API_KEY` | No | callmebot API key for WhatsApp |

## Config File Detected

The skill looks for a `.env` template and writes secrets to it.

```bash
# Check for existing config template
ls -la .env* 2>/dev/null || echo "No template found"
```

## Step-by-Step

### Step 1 — Check for Existing Config Template

```bash
ls -la .env* 2>/dev/null || echo "No template found"
```

Expected output:
```
-rw-r--r--  1 user  staff  128 Jun  7 10:00 .env.example
No template found (if no .env exists)
```

### Step 2 — Write Config File

```bash
# Create .env file with required secrets
cat > .env << 'EOF'
# Environment Configuration
# Generated by autonomous pipeline
GITHUB_TOKEN=${GITHUB_TOKEN}
WHATSAPP_PHONE=${WHATSAPP_PHONE:-}
WHATSAPP_API_KEY=${WHATSAPP_API_KEY:-}
EOF
```

### Step 3 — Verify Config File

```bash
# Confirm the file was created and contains expected vars
ls -la .env
head -10 .env
```

Expected output:
```
-rw-r--r--  1 user  staff  128 Jun  7 10:00 .env
# Environment Configuration
GITHUB_TOKEN=<secret-hidden>
WHATSAPP_PHONE=<secret-hidden>
WHATSAPP_API_KEY=<secret-hidden>
```

## Pipeline Usage

Add this step before other pipeline steps that need secrets:

```bash
# Source the skill at the start of your pipeline
echo "=== Setting up environment ==="

# Check required secrets
if [ -z "$GITHUB_TOKEN" ]; then
  echo "ERROR: GITHUB_TOKEN not set"
  exit 1
fi

# Write config if needed
if [ ! -f .env ]; then
  cat > .env << 'EOF'
GITHUB_TOKEN=${GITHUB_TOKEN}
EOF
  echo "Created .env file"
fi

echo "=== Environment ready ==="
```

## Troubleshooting

**Problem:** `.env` file not created
**Fix:** Check that the parent directory is writable and `GITHUB_TOKEN` is set

**Problem:** Secrets show as empty in `.env`
**Fix:** Ensure secrets are registered in OpenHands → Settings → Secrets

**Problem:** GitHub API returns 401 Unauthorized
**Fix:** Verify `GITHUB_TOKEN` is valid and has required scopes
